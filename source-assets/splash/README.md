# Joel Football splash artwork

- `joel-football-option-a.png` is the selected V1 splash source. Its optimized
  1280×720 WebP derivative ships from `public/assets/splash/`.
- `joel-football-option-b.png` is intentionally preserved at full resolution
  for a future alternate splash, event screen, or loading-screen rotation. It
  is not included in the production bundle yet.

Both originals were generated with OpenAI's built-in image generation using
the established Joel character as the visual reference. Option A depicts an
airborne power kick in a bright rooftop stadium; Option B is the sunset hero
pose with elemental power ribbons.
